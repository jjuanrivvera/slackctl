## slackctl conversations create

Create a channel

```
slackctl conversations create [flags]
```

### Examples

```
  slackctl conversations create --name eng-alerts
  slackctl conversations create --name secret-plans --private
```

### Options

```
  -h, --help          help for create
      --name string   channel name (lowercase, no spaces)
      --private       create a private channel
```
