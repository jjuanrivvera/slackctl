## slackctl bookmarks remove

Remove a channel bookmark

```
slackctl bookmarks remove [flags]
```

### Examples

```
  slackctl bookmarks remove --channel C0123456 --bookmark Bk123
```

### Options

```
      --bookmark string   bookmark id to remove
      --channel string    conversation id
  -h, --help              help for remove
```
